# 项目介绍

本项目是包含多个项目的融合，通过切换branch分支，来查看对应的功能项目。其中包含，项目的搭建规范配置、项目基本配置、vue组件配置、vue+ts项目配置、和一个脚手架项目模版。本项目使用的是node18.18.0，使用的都是较新版本的包。
